Spring boot 踩坑记录

打包

[TOC]

Tests run: 1, Failures: 0, Errors: 1, Skipped: 0, Time elapsed: 2.432 s <<< FAILURE! - in com.ems.web.WebApplicationTests

解决方案

在pom.xml中添加

<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-surefire-plugin</artifactId>
    <configuration>
     <skip>true</skip>
    </configuration>
   </plugin>
